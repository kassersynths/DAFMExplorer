# Kasser Synths Color Palette

## Primary Colors

| Color | HEX | Usage |
|-------|-----|-------|
| Primary Cyan | `#00d4ff` | Main accent, titles, highlights |
| Secondary Magenta | `#ff0080` | Secondary accent, hover states |
| Tertiary Green | `#00ff88` | Success states, tertiary accent |
| Accent Purple | `#8b5cf6` | Special highlights |

## Background Colors

| Color | HEX | Usage |
|-------|-----|-------|
| Background Dark | `#0a0a0a` | Main background |
| Background Medium | `#1a1a1a` | Cards, sections |
| Background Light | `#2a2a2a` | Hover states |

## Text Colors

| Color | HEX | Usage |
|-------|-----|-------|
| Text Primary | `#ffffff` | Main text |
| Text Secondary | `#a0a0a0` | Secondary text |
| Text Muted | `#666666` | Muted text |

## Typography

- **Headers**: Bold, uppercase for emphasis
- **Body**: Clean sans-serif
- **Code**: Monospace

## Design Principles

1. **High Contrast**: Neon colors on dark backgrounds
2. **Retro-Futuristic**: 80s/90s aesthetic with modern touches
3. **Technical**: Clean, precise, synthesizer-inspired
4. **Readable**: Good contrast ratios for accessibility
